# Feature Ideas

## Idea 1 — Dark Mode for Hive Mind Graph Explorer
The graph explorer currently uses a light background. A dark mode would make it easier to read large graphs and is a frequently requested UI improvement.

## Idea 2 — Bulk Node Import via CSV
Allow users to import multiple context nodes at once via a CSV file upload. Each row becomes a node. Columns map to address, content, tags, lobe etc. Useful for migrating large amounts of existing knowledge.

## Idea 3 — Node Relationships / Graph Edges UI
Currently nodes exist but relationships between them are not visible in the UI. A visual edge editor would let users draw connections between nodes — "this decision depends on this guideline" etc.

## Idea 4 — Bee Chat History Export
Allow users to export the full chat history of any bee as a PDF or markdown file. Useful for compliance, auditing, or sharing a bee's reasoning with stakeholders outside HiveBoss.

## Idea 5 — API Usage Analytics
A simple analytics page showing API usage per bee, per workflow, per connector — how many calls were made, total honey spent, error breakdown. Helps org admins understand where costs are coming from.

## Idea 6 — Webhook on Workflow Completion
Trigger a webhook when a workflow finishes — success or failure. Lets teams integrate HiveBoss completions into Slack, Zapier, or their own systems without polling.
